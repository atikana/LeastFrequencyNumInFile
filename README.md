# LeastFrequencyNumInFile
C# challenge for Uken
1. Clone the repo/project into your directory
2. Open the repo/project in microsoft visual studio or double click "LeastFrequencyNumInFile.sln" to open
3. Inside "Solution Explorer" tab locate "Program.cs" inside LeastFrequencyNumInFile
4. Locate the "Debug" menu on the top then click start debugging
4  Verify the result in the popup debug console
5. For adding more text file (.txt), please drop the files in the "src" folder in the root directory
